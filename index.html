CSS Shapes

These are generic shapes created solely using CSS. The purpose is to provide sets of shapes that can be used as icons and other user-interface elements to replace images. The idea is to improve performance and simplify design by replacing icon image files with CSS equivalents. Not only does each one save an HTTP request, but it also obviates the need to do anything special to make the graphics retina-quality or responsive.
